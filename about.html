About TipCalc

TipCalc is a simple, efficient, and user-friendly tool designed to help you calculate your restaurant tips and split bills with ease. Whether you're dining out with friends or just need a quick percentage calculation, TipCalc is here to help!

Features

  • Calculate tip amount and total bill easily
  • Split the bill among multiple people
  • Store and review recent calculations using localStorage
  • Preset tip percentage buttons for fast access

Technologies Used

  • HTML5, CSS3, JavaScript
  • Bootstrap 5 & Bootstrap Icons
  • Local Storage API

Developed By

This app was developed as a beginner-friendly JavaScript project. It demonstrates key web development concepts including DOM manipulation, event handling, and data persistence using local storage.
